# Keyboard Shortcuts

## Main window

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl+O` | Open file |
| `Cmd/Ctrl+W` | Close window / go back |
| `Cmd/Ctrl+,` | Open settings |
| `Cmd/Ctrl+Shift+T` | Toggle theme |

## Whisper (voice to text)

| Shortcut | Action |
|----------|--------|
| `Alt+Space` | Open/close recording window (global) |

The global shortcut works even when Arandu is not in focus. It can be customized in **Settings → Voice to Text**.

## Block comments

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl+Click` | Add comment to block |

## Workspace

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message to agent |
| `Shift+Enter` | New line in message |

## Customizing shortcuts

Global shortcuts (such as the Whisper shortcut) can be configured in **Settings → Voice to Text → Recording Shortcut**.
