Features

GitHub Flavored Markdown

Tables, task lists, strikethrough, and autolinks — all rendered faithfully.

Syntax Highlighting

Code blocks highlighted by highlight.js with automatic light and dark themes.

Live Reload

Watches your file and refreshes instantly when you save from any editor.

Outline Navigation

Sidebar with all headings. Click to smooth-scroll to any section.

Theme Switching

Cycle between system, light, and dark themes. Your preference is remembered.

Cross-Platform

Native Swift app on macOS. Tauri-powered app on Linux and Windows.
